# WSR Chat Integration

## Setup

Create a virtual environment:
```sh
python3.13 -m venv .venv
pip install -r requirements.txt
```

## Run Jupiter Notebook

Use an IDE to run the cells.

## Run Program

```sh
python src/robot_behavior.py
```

To interact with the programm, use the following keystrokes:

- [s]earch
- [e]ngage
- escort[a]gree
- escort[d]isagree
- [h]andoff

## Print Graph

```sh
python src/graph.py
```
